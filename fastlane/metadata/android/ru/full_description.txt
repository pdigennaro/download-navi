# Features

* Свободное и Open Source программное обеспечение (FOSS)
* Material Design
* Множество тем (светлая, тёмная, чёрная)
* Поддержка Android TV
* Поддержка Chrome OS
* Пауза и возобновление загрузок
* Поддержка браузеров
* Встроенный браузер
* Одновременные загрузки (по умолчанию 3)
* Сегментная загрузка (максимум 16)
* Скачивание файлов в фоне
* Опция "Только безлимитные соединения" (например Wi-Fi)
* Опция роуминга
* Управление User agent
* Вычисления hash-значения файла (MD5 и SHA-256)
* Управление питание и контроль заряда аккумулятора
* Различные автоматические действия после окончания загрузки
* Ограничение скорости 
* [Android 8.0+] Авто распаковка архивов. Форматы архиваторов: zip, tar, 7z, ar, cpio, dump, rar (< v5). Форматы упаковщика: gzip, bzip2, xz, lzma, Pack200, DEFLATE, DEFLATE64, Z
